Step-by-Step process#
The following is a process on how to navigate and access the components in the environment. Using Putty for Linux Machines
When you open Putty, you will see all the saved sessions. Simply select the machine you wish to interact with, click Load, and then click Open. SSH keys have been distributed to all the Linux endpoints so that you will not need to log in or anything, just click and go.
Basic Connectivity Guidelines#
To navigate throughout the environment:
- Google Chrome will be used to access the web consoles of CrowdStrike as well as other supporting tools
- Putty will be used to SSH to Linux machines
- Remote Desktop will be used to RDP to windows machines
Using Putty for Linux Machines
When you open Putty, you will see all the saved sessions. Simply select the machine you wish to interact with, click Load, and then click Open. SSH keys have been distributed to all the Linux endpoints so that you will not need to log in or anything, just click and go.
Using RDP for Windows Machines
Next, we access windows machines using RDP via Remote Desktop Connection Manager. When you open it, you see the RDP sessions are organized by headquarters subnet and operations subnet. To connect to a machine, simply double-click on it and it will open. You can also right click on the subnet name and select “Connect Group” to launch RDP sessions to all of them.
Accessing CrowdStrike#
Open Google Chrome from the Jumpbox.
By default, Google Chrome will redirect you to the CrowdStrike Falcon platform login. If not, click the "Crowdstrike Falcon" bookmark.
Enter wwt.xdr.lab@wwtatc.com as the username and click "Next".
